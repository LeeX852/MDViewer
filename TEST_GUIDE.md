# MDViewer 测试指南

## 打包版本位置

打包后的应用已生成在 `dist/` 目录：

1. **免安装版本**: `dist/win-unpacked/MDViewer.exe` (115MB)
   - 可直接运行，无需安装
   - 适合快速测试

2. **安装程序**: `dist/MDViewer Setup 0.1.0.exe` (115MB)
   - 需要安装
   - 会在开始菜单创建快捷方式

## 测试项目

### 1. 窗口控制按钮
打开应用后，测试右上角三个按钮：
- [ ] **最小化按钮** (最小化窗口图标)
- [ ] **最大化按钮** (方块图标，点击后窗口应最大化，图标变为重叠方块)
- [ ] **关闭按钮** (X 图标)

### 2. 文件对话框
在菜单栏点击 "File"：
- [ ] **New** - 新建文件
- [ ] **Open...** - 应弹出文件选择对话框
- [ ] **Save** - 如果未保存过，应弹出保存对话框
- [ ] **Save As...** - 应弹出保存对话框


### 3. 关于对话框
在菜单栏点击 "Help" → "About MDViewer"：
- [ ] 应弹出原生对话框，显示版本信息、Electron 版本等

### 4. 键盘快捷键
- [ ] `Ctrl+N` - 新建
- [ ] `Ctrl+O` - 打开文件
- [ ] `Ctrl+S` - 保存
- [ ] `Ctrl+Shift+S` - 另存为

## 预期问题

如果在生产环境（打包版本）中发现问题：
1. 查看是否有任何错误弹窗
2. 尝试打开开发者工具 (如果可用)
3. 检查是否有调试日志输出
